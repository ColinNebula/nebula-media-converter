# 🌟 Nebula Media Converter

> **Transforming media files with the power of the cosmos**

## 🚀 Quick Start

### Prerequisites
- Node.js 16.0 or higher
- npm or yarn package manager
- Modern web browser with WebAssembly support

### Installation

```bash
# Clone the repository
git clone https://github.com/Nebula3D-Dev/nebula-media-converter.git

# Navigate to project directory
cd nebula-media-converter

# Install dependencies
npm install

# Start development server
npm start
```

The application will open at `http://localhost:3000`

### Production Build

```bash
# Create optimized production build
npm run build

# Deploy to your preferred hosting platform
npm run deploy
```

## 🎯 Usage Guide

### **Basic Conversion**
1. **Upload Files**: Drag and drop or click to select files
2. **Choose Format**: Select your desired output format
3. **Configure Settings**: Adjust quality and compression (Premium)
4. **Convert**: Click convert and watch the magic happen
5. **Download**: Get your converted files instantly

### **Premium Features**
- **Batch Processing**: Convert multiple files simultaneously
- **Advanced Settings**: Custom bitrates, resolutions, and codecs
- **Priority Processing**: Faster conversion speeds
- **Extended Storage**: Keep files longer in cloud storage

### **Admin Access**
- **Keyboard Shortcut**: `Ctrl + I` (hidden from UI)
